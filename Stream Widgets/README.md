# ErinsMod Stream Widgets

I was considering making an OBS plugin for ErinsMod, but, I noticed OBS has an internal browser source, and a plugin was totally unnecessary!

## How to use:

### 1. Enable OutGauge API

<img src="https://github.com/ErinSteph/ErinsMod/blob/main/Stream%20Widgets/enable_og.png?raw=true" />

### 2. Go to widgets menu and open customiser

<img src="https://github.com/ErinSteph/ErinsMod/blob/main/Stream%20Widgets/widget_menu.png?raw=true" />

<img src="https://github.com/ErinSteph/ErinsMod/blob/main/Stream%20Widgets/open_customiser.png?raw=true" />

### 3. Select widget, customise, copy URL and CSS

<img src="https://github.com/ErinSteph/ErinsMod/blob/main/Stream%20Widgets/customise_widget.png?raw=true" />

### 4. Open OBS and add a browser source

<img src="https://github.com/ErinSteph/ErinsMod/blob/main/Stream%20Widgets/add_source.png?raw=true" />

### 5. Enter URL and CSS

<img src="https://github.com/ErinSteph/ErinsMod/blob/main/Stream%20Widgets/add_settings.png?raw=true" />

## Done!

Now you can set it up how you like it :)
